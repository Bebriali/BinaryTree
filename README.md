# 🌳 Binary Tree: Expression Solver & Differentiator

Учебный проект по реализации бинарного дерева для парсинга, упрощения и дифференцирования математических выражений.  
Проект включает в себя полноценный лексический анализатор (токенизатор) и парсер, работающий по методу рекурсивного спуска.

---

## 💡 Идея проекта

Основная задача — превратить текстовое математическое выражение (например, `x^2 + sin(x)`)  
в древовидную структуру (AST — Abstract Syntax Tree), с которой можно выполнять:

- **Символьное дифференцирование** — вычисление производной функции  
- **Упрощение (Simplification)** — свертка констант и удаление нейтральных элементов  
- **Визуализация** — генерация дерева через GraphViz  
- **Serialization** — сохранение и загрузка дерева  

---

## 🛠 Технологический стек

- **Язык:** C++  
- **библиотеки:** TxLib.h (для Windows-версии)  
- **Визуализация:** GraphViz (`dot`)  
- **Сборка:** Makefile (GCC/G++)  

---

## 📂 Структура проекта

```
src/                — исходный код  
  tokenizer.cpp     — токенизация  
  tree_builder.cpp  — парсер (рекурсивный спуск)  
  differentiator.cpp — дифференцирование  
  tree_dump.cpp     — генерация .dot файлов  

grammar/            — описание грамматики  
include/            — заголовочные файлы  
log/                — дампы и логи  
```

---

## 🚀 Запуск проекта

### 1. Требования

Для полной функциональности требуется установленный GraphViz.

---

### 2. Сборка

```bash
make
```

---

### 3. Использование

```bash
./tree log/session_log.html
```

После запуска введите математическое выражение.

⚠️ Важно: выражение должно заканчиваться символом `$`

Пример:

```text
10*(3+(10-(4*6)))-1)$
```

---

### 4. Очистка и дополнительные команды

```bash
make clean   # удаление объектных файлов
make show    # открыть лог (Windows)
```

---

## 📝 Грамматика

Парсер реализует следующую иерархию:

- **GetG** — начало выражения  
- **GetE** — сложение/вычитание  
- **GetT** — умножение/деление  
- **GetP** — степень  
- **GetN / GetV / GetF** — числа, переменные, функции  

---

## 📊 Визуализация дерева

Программа генерирует `.dot` файлы, которые можно преобразовать в изображения:

```bash
dot -Tpng log/tree.dot -o tree.png
```

Это позволяет наглядно отслеживать изменения дерева после операций.
